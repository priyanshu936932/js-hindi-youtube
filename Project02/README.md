# Project 02 - BMI Calculator

A simple BMI (Body Mass Index) calculator built with HTML, CSS, and JavaScript. It takes weight (kg) and height (cm) as inputs and shows the calculated BMI with basic guidance.

## Features
- Input fields for weight and height
- BMI calculation on button click
- Result display with BMI value
- Basic styling for layout

## Files
- index.html – page markup
- style.css – styling
- chaiandcode.js – BMI calculation logic
- projectset2.md – project notes

## How to Use
1. Open index.html in a browser.
2. Enter your weight in kg and height in cm.
3. Click **Calculate** to see your BMI.

## BMI Formula
$$\text{BMI} = \frac{\text{weight (kg)}}{(\text{height (m)})^2}$$

## Notes
- Enter valid positive numbers for accurate results.
- Height is expected in centimeters and weight in kilograms.
